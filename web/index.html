Public Docker Hub Secret Scanner

Layer-aware scanning for real image history.

layerleak inspects OCI manifests, config metadata, final files, and deleted-layer artifacts so findings keep the provenance that flat blob scanning usually drops.

What ships today

Scanner CLI

Digest-first scans for public Docker Hub images without a local Docker daemon.

Postgres persistence

Deduplicated findings and occurrences stored by manifest digest and fingerprint.

Minimal API

Read endpoints plus a synchronous scan endpoint for a Postgres-backed instance.

Why it feels different

Designed around OCI internals instead of opaque image blobs.

History-aware

Deleted-layer files still matter. layerleak keeps them visible instead of flattening them away.

Digest-first

Tags are mutable labels. The scanner resolves them to immutable digests before attaching findings.

Secret-safe defaults

CLI and API output stay redacted while local findings files and Postgres preserve raw material deliberately.
